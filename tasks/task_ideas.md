## Task Ideas
The plan is to have different categories of tasks, with varying difficulties. Most will ask to implement code using Python (might add SQL tasks in the future).

*Some of these tasks will not be used


### DSA Tasks (Data structures & Algorithms)

#### 1. Valid Palindrome (easy)
Write a function to check if a string is a palindrome, ignoring all non-alphanumeric characters and case 

#### 2. Two Sum (easy)
Write a function that returns the indices of two numbers that sum up to a target number

#### 3. Binary Search (easy/medium)
Write a function that returns the index of the target number in an array of integers. The array is sorted in ascending order

...

### Data Processing Tasks

#### 1. Remove Duplicate Records
Write a function that removes duplicate records from a list of dictionaries

#### 2. Find top N customers

Write a function that returns the top N customers by total spending
